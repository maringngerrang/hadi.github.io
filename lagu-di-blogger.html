Memasang Lagu di Blogger

Halaman ini merupakan halaman demo tentang cara memasang lagu sendiri di Blogger. Kamu bisa menggunakan contoh source code di bawah dan memasangnya di blog kamu atau silakan buka di sini untuk informasi lebih lanjut. Bagian yang ditandai adalah link dari lagu yang akan diputar.

Contoh lagu yang autoplay
Source code: 
<audio controls="" autoplay="">
    <source src="http://upload.wikimedia.org/wikipedia/commons/6/63/Corelli-Trio_Sonata_11-1.ogg" />
    Jika anda tidak bisa melihat kontrol audio, browser anda tidak mendukung
</audio>
Contoh lagu yang loop (berulang)
Source code: 
<audio controls="" loop="">
    <source src="http://upload.wikimedia.org/wikipedia/commons/3/3c/01_-_Vivaldi_Spring_mvt_1_Allegro_-_John_Harrison_violin.ogg" />
    Jika anda tidak bisa melihat kontrol audio, browser anda tidak mendukung
</audio>

Berikut ini adalah contoh embed lagu yang menggunakan file lagu dari hasil download. Jika tidak bisa dimainkan maka link sudah kadaluwarsa.
